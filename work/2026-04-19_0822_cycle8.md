## 5. Health Monitoring Plan (P2 Optimization)
Based on current checks, a minimal health script could be:
```bash
#!/bin/bash
# sentinel-health.sh
SERVICES=("keeper:8900" "agent-api:8901" "holodeck:7778" "seed-mcp:9438" "shell:8846")
for service in "${SERVICES[@]}"; do
    name="${service%:*}"
    port="${service#*:}"
    if [[ $name == "holodeck" ]]; then
        nc -z localhost $port && echo "$name: UP" || echo "$name: DOWN"
    else
        http_code=$(curl -s -o /dev/null -w "%{http_code}" http://localhost:$port/health 2>/dev/null)
        [[ $http_code == "200" ]] && echo "$name: UP ($http_code)" || echo "$name: DOWN ($http_code)"
    fi
done
```
This would run every 60 seconds, logging to a timestamped file. Alerts could trigger on two consecutive failures.
